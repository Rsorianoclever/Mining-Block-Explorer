# Mining-Block-Explorer
Un conjunto de herramientas en Python para la visualización de modelos de bloques en 3D, la generación de histogramas de elementos clave y la creación de curvas tonelaje-ley. Ideal para profesionales de la minería y la geología que deseen analizar datos geoespaciales de manera eficiente

Este programa en Python es una herramienta  para la visualización de modelos de bloques en 3D, así como para la generación de histogramas de elementos de interés en el modelo y la creación de curvas tonelaje-ley. Es especialmente útil en el ámbito de la minería y la geología para el análisis de datos geoespaciales.

## Características Clave

- **Visualización en 3D**: Explora modelos de bloques en tres dimensiones para comprender la distribución espacial de los elementos de interés.

- **Histogramas Interactivos**: Genera histogramas personalizables de los elementos clave en el modelo, lo que facilita el análisis de la distribución de datos.

- **Curvas Tonelaje-Ley**: Crea curvas de tonelaje-ley para evaluar la calidad del depósito y su potencial económico.

## Cómo Usar

1. Clona este repositorio en tu máquina local.

2. Ejecuta el programa en tu entorno de desarrollo de Python.

3. Carga tu modelo de bloques en el formato compatible (por ejemplo, CSV o XYZ).

4. Utiliza las opciones de visualización 3D, histogramas y curvas tonelaje-ley según tus necesidades.
